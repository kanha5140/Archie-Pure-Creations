## 💎 The Vision
Archie Pure Creations is more than a store; it is a digital atelier. This project bridges the gap between traditional craftsmanship and modern automation. Built for high-end handmade jewelry and crafts, the platform focuses on a "Slow Art" philosophy—elevating the user experience through elegant design and seamless business workflows.

---

## 🚀 Key Features

* **Premium UX/UI**: A bespoke aesthetic utilizing *Cormorant Garamond* and *Montserrat* typography to reflect luxury brand values.
* **Headless CMS Integration**: Real-time product management powered by **Google Sheets**, allowing for instant inventory updates without code changes.
* **Automated Order Processing**: Direct-to-Telegram bot integration that sends itemized invoices and customer data instantly.
* **Intelligent Parcel Tracking**: A mobile-optimized tracking portal featuring a CSS-animated timeline and real-time status fetching from logistics sheets.
* **Optimized Performance**: Lightweight architecture ensuring fast load times and a fluid mobile-first experience.

## 📱 Mobile-First Logistics
The tracking system is designed for the modern user on the go.
* **Animated States**: Features "Please Wait" loading spinners for professional feedback.
* **Detailed Insights**: Provides customer-specific data including Order ID, Status, and Estimated Delivery.
* **Responsive Timeline**: A vertical CSS timeline that adapts to any screen size for high readability.
